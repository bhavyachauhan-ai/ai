# what is rag?

- :contentReference[oaicite:0]{index=0} (RAG) is a method used in AI to improve answers by combining:
  - searching for information (retrieval)
  - generating answers (generation)

# simple meaning

- instead of only using what the AI already knows
- RAG allows the AI to:
  - look up fresh or external data
  - then create a better and more accurate answer

# how it works (step by step)

- user asks a question
- system searches a database or documents
- relevant information is retrieved
- AI uses that information to generate the final answer

# example

- normal AI:
  - answers based only on training data

- RAG AI:
  - checks documents, PDFs, or websites
  - then gives a more updated and precise answer

# why rag is useful

- gives more accurate answers
- reduces wrong or made-up information
- allows use of private data (like company files)
- keeps answers up-to-date

# short summary

- RAG = search + generate
- makes AI smarter and more reliable
